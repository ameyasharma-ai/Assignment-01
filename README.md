# TuteDude HTML Learning Demo Page

This is a comprehensive HTML webpage created as part of **Assignment 01** to demonstrate proficiency in semantic HTML structures and modern web design.

## 🔗 Project Links

- **GitHub Repository:** [https://github.com/ameyasharma-ai/Assignment-01](https://github.com/ameyasharma-ai/Assignment-01)
- **Live Deployment:** [https://assignment-01-eight-alpha.vercel.app](https://assignment-01-eight-alpha.vercel.app)

## 📝 Project Overview

The webpage is built for a fictional learning platform named **TuteDude** and includes:
- A responsive, modern design leveraging CSS glassmorphism, gradients, and custom fonts.
- Proper use of semantic HTML tags (`<header>`, `<main>`, `<article>`, `<aside>`, `<footer>`, etc.).
- Complex tables featuring nested rows for a cafeteria menu.
- A structured feedback form utilizing various input types.
- Embedded interactive media (`<audio>` and `<video>`).

## 🛠️ How to View Locally
Simply clone this repository and open `index.html` in your favorite web browser.
